**Project Period:** 2022.09 – 2023.12
**Funding:** Beijing Municipal Science & Technology Plan (北京市科技计划)

## Overview

Designed and developed a humanoid robot system with autonomous mobility and dexterous manipulation capabilities, featuring innovative wheel-legged transformation for enhanced locomotion versatility.

## Key Contributions

- Built **real-time software architecture** based on RTOS and EtherCAT (0.25–1kHz control frequency)
- Designed **minimal-modification wheel-legged transformation** with active and passive wheel fusion
- Proposed **phase-decomposition-guided PSO trajectory optimization** for stable mode switching
- Published **1 first-author paper** in Q1 TOP journal

## Technical Highlights

- Multi-process communication architecture with shared memory
- Force-position hybrid control at 0.25–1kHz
- Upper body–ankle-foot cooperative compliant control
- Motor selection, verification, and electromechanical-control integration
