## Overview

A comprehensive, bilingual (English/Chinese) educational resource for beginners learning CoppeliaSim (formerly V-REP) robotics simulation. Includes slide presentations, annotated code examples, and ready-to-use simulation models.

## Key Features

- **Bilingual content**: complete English and Chinese versions of all materials
- **Structured curriculum**: from basic scene setup to advanced robot control
- **Slide presentations**: lecture-ready PPTX files for teaching
- **Code examples**: MATLAB scripts with detailed comments
- **Simulation models**: pre-built robot scenes for hands-on practice

## Technical Highlights

- Covers CoppeliaSim API (Lua scripting + MATLAB remote API)
- Progressive difficulty: basic → intermediate → advanced robotics simulation
- Designed for both self-study and classroom use

## My Role

Author and maintainer — created all tutorial content, slides, code examples, and simulation models as part of the CoppeliaSim Edu Kit internationalization project.
